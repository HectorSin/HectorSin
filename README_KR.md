도메인 특화 콘텐츠를 위한 커스텀 프롬프팅 전략과 Gemini API 통합 구현

**🔧 기술 하이라이트:**

지능적인 뉴스 수집 및 큐레이션을 위한 Perplexity API 활용

자연스러운 한국어 음성 합성을 위한 Naver Clova TTS 통합

확장 가능한 서비스 제공을 위해 GCP Cloud Run에 배포된 FastAPI 백엔드 구축
